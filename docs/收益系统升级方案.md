# 收益系统升级方案

> 状态：已拆分为阶段文档。入口名称固定使用 **股息日历**。

这次升级的目标是：在现有「当前持仓 + 年化预估」基础上，新增一层 **历史事件账本**，支撑已收股息、即将到账股息、年度收益、上一年对比和历年对比。

## 底部导航

最终底部导航保留 3 个入口：

1. **资产**：现有首页，保留当前持仓、仓位结构、市值、预估年化股息。
2. **收益汇总**：年度复盘，展示股息收入、资金收益、上一年同比、历年趋势。
3. **股息日历**：当年股息现金流，展示今年已到账、即将到账、预计全年、按月/按股分布和逐笔明细。

## 总体阶段

- **P1a · 数据地基**：扩展私有 `portfolio.json`，扩展公开 `market.json` 的逐笔派息，增加打开 App 时的自动结算。
- **P1b · 股息日历**：增加底部导航和股息日历页面。
- **P2 · 收益汇总**：增加年度收益页，支持上一年和历史年份对比。
- **P3 · 自动化与增强**：到账确认、历史回填、私有定时结算、交易流水、再投资归集、公告数据源。
