# 02 · P1b 股息日历

## 目标

新增底部导航，并实现 **股息日历** 页面。

本阶段完成后，用户可以清楚看到：

- 今年已到账股息。
- 今年即将到账股息。
- 预计全年股息。
- 按月分布。
- 按股票分布。
- 每笔股息明细。

## 前置条件

需要先完成：

- [01-P1a-数据地基.md](./01-P1a-数据地基.md)

如果 P1a 尚未完成，本阶段不要开始。

## 页面入口

底部导航：

```text
资产 | 收益汇总 | 股息日历
```

本阶段可以先让「收益汇总」显示空状态或占位页，但入口可以先建好。

## 股息日历页面结构

### 顶部指标

展示：

- 今年已到账
- 即将到账
- 预计全年
- 预估年化股息

说明：

- 今年已到账来自 `dividendLedger`。
- 即将到账 = `pending` + `forecast`。
- 预计全年 = 已到账 + 即将到账。
- 预估年化股息来自现有 TTM 逻辑，用于并列对照。

### 筛选

提供三段筛选：

```text
全部 / 核心仓 / 打工仓
```

筛选影响：

- 顶部指标
- 按月分布
- 按股票分布
- 明细列表

### 即将到账

展示两类：

- 已除息待确认：来自 `dividendLedger`，`receiptStatus="pending"`。
- 节奏预估：从历史派息节奏推算，只展示，不写入 ledger。

`forecast` 必须有明显标记，避免和确定股息混淆。

### 按月分布

按除息日月份统计。

每个月展示：

- 已到账金额
- 待到账金额
- 预估金额

### 按股票分布

按 symbol 聚合当年股息。

展示：

- 股票名称
- 已到账
- 待到账
- 预估
- 占比

### 每笔明细

字段建议：

- 股票
- 除息日
- 每股股息
- 持股数
- 税后 CNY
- 状态
- 可信度

状态文案建议：

- 已到账
- 待确认到账
- 节奏预估
- 当前股数估算
- 快照推算

## 首页文案调整

现有首页的「年度股息总金额」改名：

```text
预估年化股息
```

不要改变原计算逻辑。

## 隐私模式

隐私模式需要遮挡：

- 已到账金额
- 即将到账金额
- 预计全年
- 每笔明细金额

可以保留日期、股票名称、状态。

## 不做的事

本阶段不做：

- 年度收益汇总的完整计算。
- 出入金编辑。
- 历史年份手动回填。
- 到账确认编辑 UI。
- 交易流水。

## 验收标准

- 底部导航可以在「资产 / 收益汇总 / 股息日历」之间切换。
- 资产页现有能力不退化。
- 首页股息文案改为「预估年化股息」。
- 股息日历能基于 `dividendLedger` 正确汇总当年股息。
- 空数据时有合理空状态。
- 隐私模式生效。
- 移动端布局不溢出。
- `npm run build` 通过。
