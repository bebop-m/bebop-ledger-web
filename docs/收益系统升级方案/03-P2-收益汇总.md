# 03 · P2 收益汇总

## 目标

实现 **收益汇总** 页面，用年度口径回答：

- 今年股息收入是多少？
- 今年资金收益是多少？
- 和上一年相比如何？
- 和历史年份相比如何？
- 股息收入是否在持续增长？

- [01-P1a-数据地基.md](./01-P1a-数据地基.md)
- [02-P1b-股息日历.md](./02-P1b-股息日历.md)

## 页面结构

### 顶部年度概览

默认展示当前年。

指标：

- 股息收入
- 资金收益
- 合计参考
- 较上一年变化

合计参考必须带口径说明：

```text
合计为参考值；复利再投资和现金账户未完整跟踪前，股息与资金收益不作为唯一总收益口径。
```

### 年度列表

每年一行：

- 年份
- 股息收入
- 资金收益
- 合计参考
- 净注入
- 年末净值
- 同比

### 历年趋势

展示：

- 股息收入线
- 资金收益线
- 可选合计参考线

### 北极星指标

单独展示：

```text
核心仓股息收入
```

这是衡量「让股息收入逐渐增加」的核心指标。

## 计算口径

### 股息收入

```text
股息收入(Y) = sum(dividendLedger.netCny where exDate.year == Y)
```

若某年没有 ledger，但有 `yearlyManual.dividendCny`，使用手动回填值。

### 资金收益

```text
资金收益(Y) = 年末净值 - 年初净值 - 当年净注入
```

其中：

```text
当年净注入 = deposit - withdraw
```

年初净值可以取上一年年末净值。

如果快照缺失：

- 优先使用 `yearlyManual.yearEndNetCny`。
- 仍缺失则显示为空或「待回填」。

### 同比

```text
同比 = (今年值 - 去年值) / abs(去年值)
```

去年值为 0 或缺失时，不显示百分比，显示「无可比数据」。

## 筛选

收益汇总支持：

```text
全部 / 核心仓 / 打工仓
```

股息收入可以按 bucket 精确筛选。

资金收益 v1 如果没有交易流水，不能可靠按 bucket 拆分；可以：

- 全部口径显示资金收益。
- bucket 筛选下只拆股息收入。
- UI 明确说明资金收益暂按全账户口径。

## 不做的事

本阶段不做：

- 完整交易流水。
- 再投资正确归集。
- 私有仓库定时结算。
- 公司公告派息数据源。

## 验收标准

- 收益汇总入口可用。
- 当前年有年度概览。
- 历史年份有列表和趋势。
- 无历史净值时不报错，并提示待回填。
- 股息收入按除息日统计。
- 合计参考有清晰口径说明。
- 核心仓股息收入作为独立指标展示。
- 隐私模式遮挡金额。
- `npm run build` 通过。
