# 单字段编辑抽屉 · 施工方案（基准股数/税率/每股股息/负债 四弹窗共用）

入口：持仓行四个可点域 + 首页净资产 label。渲染：modal.js quantity/tax/dividend/liability 分支，共用同一套模板样式。

## 抽屉通用规格（本界面为底部抽屉）
- 顶部圆角 22px；顶端把手 32×3px 圆角 2px，色 var(--track)；背景 var(--bg)；无边框。
- 遮罩：日间 rgba(59,54,46,.28)，夜间 rgba(0,0,0,.45)；背景页保持原位被压暗。
- 动作行：居中排布、间距 46px 的纯文字键——删除=var(--up)、取消=var(--hint)、保存=var(--ink) 且下缀 4px 金点（::after）。严禁色块底按钮。
- 标题：11px 字距 5px var(--label) 居中；说明行 11px var(--muted)。

## 内容结构
1. 居中标题（如「基准股数」）11px 字距 5px label 色。
2. 两行口径说明 11px muted 居中（沿用现有 note 文案，如「当前持股 = 基准 + 之后的交易回放」）。
3. **金线大输入**：34px/600 数字居中，底部 2px var(--gold) 托线，单位（股/%）13px muted 后缀。input 隐藏原生边框。
4. 动作：取消 / 保存（金点）。税率**纯手填**，不加快捷选项。
