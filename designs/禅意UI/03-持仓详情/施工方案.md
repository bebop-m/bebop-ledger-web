# 持仓详情抽屉 · 施工方案

入口：持仓页逐股行点名称。渲染：src/modal.js renderHoldingDetailModal。

## 抽屉通用规格（本界面为底部抽屉）
- 顶部圆角 22px；顶端把手 32×3px 圆角 2px，色 var(--track)；背景 var(--bg)；无边框。
- 遮罩：日间 rgba(59,54,46,.28)，夜间 rgba(0,0,0,.45)；背景页保持原位被压暗。
- 动作行：居中排布、间距 46px 的纯文字键——删除=var(--up)、取消=var(--hint)、保存=var(--ink) 且下缀 4px 金点（::after）。严禁色块底按钮。
- 标题：11px 字距 5px var(--label) 居中；说明行 11px var(--muted)。

## 内容结构
1. 抬头：左「代码 · 所属仓」10px hint 上、公司名 18px/800 下；右权重 13px/700 muted。
2. 主数：居中「当前持股」label + 30px 数字 +「股」单位 13px muted。
3. 台账七行（12px，label muted 左 / 值 ink 700 右，行距 13px）：现价（原币）、持仓市值、交易起点基准股数、股息税率（未设显示「未设置（按 0% 估算）」）、每股 TTM 股息、税前年化股息、税后年化股息。
4. 沉底说明 10px hint 居中：「自动行情/手动每股股息 · 金额按当前汇率折算人民币 · 已除息事件以除息日快照为准」。
5. 动作：仅「关 闭」。
