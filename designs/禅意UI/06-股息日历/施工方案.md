# 股息日历页 · 施工方案

以 定稿图.html 为唯一视觉标准。

## 结构（自上而下）
1. 页头 grid：中「股息日历」字距页名；无 rule。
2. 三仓筛选居中（全部/核心仓/打工仓），选中=ink+**4px 金点下划**（不是下划线条）。
3. **居中 hero**：label「预计全年」→ 38px 主数 → 同比行「+x.x% · 较去年」（YoY 数据沿用现有 projectedYoy）。
4. 三段构成线：2px、宽 280px 居中——**实色**金(已到账)/墨(在途)/纸(预估)，段宽=金额占比；下挂三行图例（4px 色点+名+口径小字+金额右对齐）。三桶互斥恒等于全年（现有 pipeline 口径）。**删除紫色斜纹带与旧「月度股息」12 柱图**。
5. 12 月点阵（6 列 2 行）：月号+金额(9.5px)+3.5px 金点(有派息)。**三态**：已过月整体 opacity .42；当月 ink 加粗+金点 5px；未到正常。**点月打开月明细抽屉**（唯一月明细入口）；点阵下提示行「点按月份查看当月逐笔并确认到账」。
6. 列表：「待确认 · N 笔」置顶（金额右 aside；行=日期+公司 ink | 金额+「待确认」var(--up)）；「近期」倒序（已到账=金、在途=hint）。
