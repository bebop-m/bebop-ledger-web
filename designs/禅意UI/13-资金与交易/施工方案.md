# 资金与交易页 · 施工方案

以 定稿图.html 为唯一视觉标准。

## 结构
1. 页头 grid：中「资金与交易」。
2. 居中 hero：label「{年} · 净注入」→ 38px 带符号主数 → 「入金 ¥x · 出金 ¥x · N 笔」。
3. **现金余额次级焦点（居中）**：label 字距「现金余额」→ 20px 值 → 「基准日 MM-DD · 点击校准」（校准两字金色），点击开 openingCash 弹窗。未设置时值显示「未设置」。
4. 四类计数一行居中 10.5px hint：买入/卖出/出入金/股息。
5. 三段流水（买卖/出入金/股息入账）：节标+右笔数；行=日期+类型词（买=up/卖=down，入金出金 ink）+公司/备注 detail 小行 | 右带符号金额 **墨色**；每段默认 3 行+「展开全部 N 笔」居中文字。行点击进对应编辑抽屉。
